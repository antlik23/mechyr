# Ovládni Měchýř App

Ovládni Měchýř is a companion app to the OvládniMěchýř website, designed to help users keep a voiding diary as part of their participation in the project.

## Dependencies

Before running the project, make sure you have the following tools installed:

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (e.g. `>=3.6.0 <4.0.0`)
- [Dart SDK] – included with Flutter
- [Android Studio](https://developer.android.com/studio) or [Xcode](https://developer.apple.com/xcode/) – required for running the app on an emulator or physical device
- (optional) [VS Code](https://code.visualstudio.com/) with the Flutter plugin for a better development experience

## Installation

```bash
flutter pub get
```

### Build and Run

```bash
open -a simulator (to get iOS Simulator)
flutter run
```
