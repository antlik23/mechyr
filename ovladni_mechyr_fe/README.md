# Ovládni měchýř FE

This is the frontend repository for "Ovládni měchýř" application.

## Getting Started

### Prerequisites

#### Node.js

You need to have [Node.js](https://nodejs.org/) version 20+ installed on your machine.

#### pnpm

You need to have [pnpm](https://pnpm.io/) version 9+ installed on your machine.

#### Environment variables

Copy the contents of `.env.example` to `.env`.

### Installation

```bash
pnpm install
```

### Usage

#### Development

To start up a local development server, execute this command:

```bash
pnpm dev
```
